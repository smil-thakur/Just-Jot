
# Just Jot

**Just Jot** is a responsive website that enables you to maintain colorful chits of notes in a highly interactive UI. It leverages local storage for seamless data persistence and offers essential features such as searching, editing, deleting, and adding notes.

## Features

- Create and manage colorful notes.
- Automatically randomized colors for an engaging visual experience.
- Seamless data storage using local storage.
- Interactive user interface for a smooth user experience.
- Search functionality to find notes quickly.
- Edit, delete, and add notes effortlessly.

## Getting Started

Follow these instructions to set up and run **Just Jot** locally on your machine.

### Prerequisites

To run this project, you need the following tools:

- [Web Browser](https://www.google.com/chrome/) - A modern web browser like Google Chrome.
- [Git](https://git-scm.com/) - Version control system for cloning the repository.

### Installation

1. Clone the repository to your local machine using Git:

```bash
git clone https://github.com/your-username/just-jot.git
```

2. Navigate to the project directory:

```bash
cd just-jot
```

3. Open the `index.html` file in your web browser:

```bash
# For Linux and macOS
xdg-open index.html

# For Windows
start index.html
```

**Just Jot** should now be up and running in your browser.

## Usage

- To create a new note, click the "Add Note" button.
- To edit an existing note, click the "Edit" icon on the note.
- To delete a note, click the "Delete" icon on the note.
- Use the search bar to find specific notes quickly.
- Enjoy the dynamic and colorful user interface.
